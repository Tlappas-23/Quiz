# JavaScript Fundamentals Quiz

An interactive quiz built with **HTML**, **CSS**, and **JavaScript** to test basic JavaScript knowledge.

## Deliverables
This quiz:
- Presents at least 5 sequential questions  
- Allows only one answer per question  
- Gives instant feedback with clear visual indicators  
- Tracks and displays the final score  
- Offers a full restart without refreshing the browser  

## Evaluation Criteria
- All features work reliably (answer locking, score tallying, final score display, restart)  
- JavaScript is organized into functions with descriptive variable names  
- Dynamic element creation replaces hard-coded HTML for questions and options  
- Buttons clearly indicate correct and incorrect answers  
- Navigation and progress text update appropriately  

## Features
- Start screen before the first question  
- Progress display showing current question out of total  
- Responsive design using Flexbox and modern CSS  
- Clean, accessible layout with color feedback for user actions  

## How to Use
1. Open `index.html` in your browser.  
2. Click **Start Quiz** to begin.  
3. Select an answer for each question.  
4. View your score at the end.  
5. Click **Restart Quiz** to play again.

## File Structure
Quiz App - Index.html - Style.css - Script.js

## Customization
- Add or edit questions in the `quizData` array in `script.js`.  
- Adjust styles and colors in `style.css`.
